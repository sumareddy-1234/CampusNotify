# 🚀 CampusNotify

🌟 Discover. Engage. Achieve – All in One Place

CampusNotify is a centralized web platform that helps students stay updated with internships, hackathons, cultural events, technical events, sports, and club activities — all in one place.

---

## 📌 Problem

Important campus updates are often scattered across multiple platforms (WhatsApp, notice boards, emails), leading to missed opportunities.

---

## 💡 Solution

CampusNotify provides a single platform where students can easily access all campus-related updates in an organized manner.

---

## ✨ Features

* Internship and placement updates
* Hackathon and technical event notifications
* Cultural and sports event listings
* Simple and user-friendly interface
* Categorized event browsing

---
